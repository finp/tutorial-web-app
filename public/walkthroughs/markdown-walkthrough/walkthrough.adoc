# A custom walkthrough written in markdown

This walkthrough demonstrates authoring in markdown. 


The actual processor assumes walkthrough is written in link:https://www.asciidoc.org[Asciidoc]. 
However, asciidoc supports most markdown syntax. See https://asciidoctor.org/docs/asciidoc-vs-markdown/[this page] for more information.

A custom walkthrough that shows users how to do:

- set up this
- integrate with that
- use the other

Not all markdown syntax is supported, the following table outlines changes required:


[#asciidoc-vs-markdown]
.AsciiDoc language features compared to Markdown (table written in asciidoc)
